Hướng dẫn xây dựng ứng dụng Todo List đơn giản bằng HTML, CSS và JavaScript

Chào các bạn, hôm nay chúng ta sẽ cùng nhau thực hành xây dựng một dự án "kinh điển" cho người mới bắt đầu học lập trình web: Ứng dụng Todo List.

Dự án này tuy nhỏ nhưng sẽ giúp bạn hiểu rõ cách 3 thành phần HTML, CSS và JavaScript phối hợp với nhau như thế nào. Chúng ta sẽ thực hiện 3 chức năng cốt lõi:

  1. Thêm công việc vào danh sách.
  2. Hiển thị danh sách công việc.
  3. Xóa công việc đã hoàn thành.

Chúng ta sẽ chia dự án thành 3 file: index.html, style.css và script.js. Hãy bắt đầu nào!

Bước 1: Xây dựng khung xương với HTML

Trước tiên, chúng ta cần tạo cấu trúc trang web. Chúng ta cần một ô nhập liệu (input), một nút bấm (button) để thêm, và một danh sách rỗng (ul) để chứa các công việc sẽ thêm vào sau này.

Mô tả mã lệnh: Dưới đây là nội dung file index.html. Tôi sẽ tạo một thẻ div bao quanh để dễ căn chỉnh. Quan trọng nhất là các thuộc tính id (như todo-input, add-btn, todo-list) để JavaScript có thể tìm và xử lý chúng sau này. 

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Todo List Đơn Giản</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <div class="container">
        <h2>Danh sách việc cần làm</h2>
        
        <div class="input-group">
            <input type="text" id="todo-input" placeholder="Nhập công việc mới...">
            <button id="add-btn">Thêm</button>
        </div>

        <ul id="todo-list">
            </ul>
    </div>

    <script src="script.js"></script>
</body>
</html>

Giải thích mã lệnh:

  • <div class="container">: Dùng để gom nhóm tất cả nội dung vào giữa màn hình.
  • <input id="todo-input">: Nơi người dùng gõ nội dung công việc.
  • <button id="add-btn">: Nút bấm để kích hoạt hành động thêm.
  • <ul id="todo-list">: Đây là thẻ danh sách không có thứ tự. Hiện tại nó đang rỗng, chúng ta sẽ dùng JavaScript để chèn các thẻ <li> vào đây.
  • <script src="script.js">: Dòng này đặt ở cuối body để đảm bảo HTML tải xong thì JavaScript mới chạy.

Bước 2: Trang trí giao diện với CSS

Một ứng dụng đẹp sẽ tạo cảm hứng hơn. Chúng ta sẽ căn giữa mọi thứ và làm đẹp cho danh sách công việc.

Mô tả mã lệnh: Trong file style.css, chúng ta sẽ thiết lập font chữ, căn giữa nội dung và tạo kiểu cho các thẻ li (đại diện cho từng công việc) để chúng trông gọn gàng, tách biệt nhau. 

/* Thiết lập cơ bản */
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
    margin: 0;
}

/* Khung chứa chính */
.container {
    background: white;
    padding: 20px;
    border-radius: 8px;
    box-shadow: 0 0 10px rgba(0,0,0,0.1);
    width: 350px;
}

h2 {
    text-align: center;
    color: #333;
}

/* Khu vực nhập liệu */
.input-group {
    display: flex;
    gap: 10px;
    margin-bottom: 20px;
}

input {
    flex: 1; /* Chiếm hết khoảng trống còn lại */
    padding: 10px;
    border: 1px solid #ddd;
    border-radius: 4px;
}

button#add-btn {
    padding: 10px 15px;
    background-color: #28a745;
    color: white;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

button#add-btn:hover {
    background-color: #218838;
}

/* Danh sách công việc */
ul {
    list-style-type: none; /* Bỏ dấu chấm tròn mặc định */
    padding: 0;
}

li {
    background: #fff;
    border-bottom: 1px solid #eee;
    padding: 10px;
    display: flex;
    justify-content: space-between; /* Đẩy nút xóa sang phải */
    align-items: center;
}

/* Nút xóa */
.delete-btn {
    background-color: #dc3545;
    color: white;
    border: none;
    padding: 5px 10px;
    border-radius: 4px;
    cursor: pointer;
    font-size: 12px;
}

Giải thích mã lệnh:

  • body { display: flex; ... }: Giúp căn toàn bộ khung .container ra chính giữa màn hình trình duyệt.
  • input { flex: 1; }: Giúp ô nhập liệu tự động giãn rộng ra để lấp đầy khoảng trống, đẩy nút "Thêm" sang sát mép phải.
  • justify-content: space-between: trong thẻ li : Đây là mấu chốt để tên công việc nằm bên trái và nút Xóa nằm bên phải.

Bước 3: Xử lý logic với JavaScript

Đây là phần quan trọng nhất. Chúng ta sẽ viết mã để khi bấm nút "Thêm", một công việc mới sẽ xuất hiện, và khi bấm "Xóa", nó sẽ biến mất.

Mô tả mã lệnh: Trong file script.js, chúng ta sẽ thực hiện 3 việc:

  1. Lấy các phần tử từ HTML (Input, Button, UL).
  2. Viết hàm addTodo để tạo ra một thẻ li mới chứa nội dung và nút xóa.
  3. Gắn sự kiện click cho nút Thêm.
// 1. Lấy các phần tử cần thiết từ DOM
const input = document.getElementById('todo-input');
const addBtn = document.getElementById('add-btn');
const todoList = document.getElementById('todo-list');

// 2. Hàm xử lý việc thêm công việc
function addTodo() {
    // Lấy giá trị người dùng nhập vào
    const text = input.value;

    // Kiểm tra: Nếu ô nhập rỗng thì không làm gì cả
    if (text.trim() === "") {
        alert("Vui lòng nhập nội dung công việc!");
        return;
    }

    // --- Bắt đầu tạo phần tử HTML mới ---

    // Tạo thẻ <li>
    const li = document.createElement('li');

    // Tạo nội dung chữ cho thẻ li
    const span = document.createElement('span');
    span.textContent = text;

    // Tạo nút Xóa
    const deleteBtn = document.createElement('button');
    deleteBtn.textContent = 'Xóa';
    deleteBtn.classList.add('delete-btn'); // Thêm class để nhận CSS

    // Gắn sự kiện xóa ngay khi tạo nút
    deleteBtn.onclick = function() {
        todoList.removeChild(li); // Xóa thẻ li này khỏi danh sách
    };

    // Ghép các phần tử lại với nhau
    li.appendChild(span);       // Bỏ chữ vào li
    li.appendChild(deleteBtn);  // Bỏ nút xóa vào li
    
    // Đưa thẻ li hoàn chỉnh vào danh sách ul
    todoList.appendChild(li);

    // --- Kết thúc tạo ---

    // Xóa trắng ô nhập liệu để nhập cái tiếp theo
    input.value = "";
    input.focus(); // Đưa con trỏ chuột quay lại ô nhập
}

// 3. Gắn sự kiện click cho nút Thêm
addBtn.addEventListener('click', addTodo);

// (Tùy chọn) Cho phép ấn Enter để thêm
input.addEventListener('keypress', function(event) {
    if (event.key === "Enter") {
        addTodo();
    }
});

Giải thích mã lệnh:

  • document.getElementById(...): Giúp JavaScript nắm giữ các thẻ HTML để điều khiển.
  • document.createElement('li'): Thay vì viết HTML, chúng ta dùng JS để tạo ra một thẻ <li> mới trong bộ nhớ.
  • li.appendChild(...): Đây là hành động "bỏ vào trong". Chúng ta bỏ đoạn text và nút Xóa vào trong thẻ <li>, rồi sau đó bỏ thẻ <li> vào trong thẻ <ul> (todoList).
  • removeChild(li): Hàm này giúp xóa bỏ phần tử con (ở đây là dòng công việc li) ra khỏi phần tử cha (todoList).
  • input.value = "": Reset ô nhập liệu về rỗng để tiện cho người dùng nhập việc tiếp theo.

Tổng kết

Vậy là chúng ta đã hoàn thành! Khi bạn chạy file index.html trên trình duyệt:

  1. Gõ chữ vào ô input.
  2. Bấm "Thêm" -> Một dòng mới xuất hiện bên dưới.
  3. Bấm "Xóa" trên dòng đó -> Dòng đó biến mất.

Bài tập này giúp bạn hiểu cốt lõi của DOM Manipulation: Tạo phần tử (createElement), Thêm phần tử (appendChild) và Xóa phần tử (removeChild).
